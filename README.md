## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/diogobruni/my-coins.git
   ```
3. Install NPM packages
   ```sh
   npm install
   # or
   yarn
   ```
4. Run as develop environment
   ```sh
   npm run dev
   # or
   yarn dev
   ```
